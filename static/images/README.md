# Изображения ресторанов

Поместите сюда изображения ресторанов:

## Требуемые файлы:

- `1.png` - Кафе «Уют»
- `2.png` - Ресторан «Вкусняшка»
- `3.png` - Sky Lounge
- `4.png` - Суши Токио
- `5.png` - Пицца Мама

## Рекомендации:
- Формат: PNG или JPG
- Размер: 600x400px (или соотношение 3:2)
- Качество: хорошее, но не слишком большой размер файла (до 500KB)

## Где взять изображения:

### Бесплатные стоки:
- https://unsplash.com/ (поиск: "restaurant", "cafe", "dining")
- https://www.pexels.com/ (поиск: "restaurant interior")
- https://pixabay.com/ (поиск: "restaurant")

### Рекомендуемые запросы:
1. Для Кафе «Уют»: "cozy cafe", "small restaurant"
2. Для Ресторан «Вкусняшка»: "italian restaurant", "fine dining"
3. Для Sky Lounge: "rooftop bar", "skyline restaurant"
4. Для Суши Токио: "sushi restaurant", "japanese restaurant"
5. Для Пицца Мама: "family restaurant", "pizzeria"

## Временные заглушки

Если у вас пока нет изображений, сайт будет работать с серым фоном на карточках ресторанов.

Для создания простых заглушек можно использовать:
- https://via.placeholder.com/600x400/7b68ee/ffffff?text=Restaurant
- https://placehold.co/600x400/7b68ee/white?text=Cafe

Просто скачайте изображения по этим ссылкам и переименуйте их в 1.png, 2.png и т.д.
